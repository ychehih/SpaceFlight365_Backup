This document provides important, late-breaking information about the Microsoft Dynamics 365 Software Development Kit (SDK)

Microsoft Dynamics 365 SDK Download Contents

The Microsoft Dynamics 365 SDK download package includes the following files and folders:

·         Bin Folder: Contains the assemblies needed for development by using Microsoft Dynamics 365

·         Resources Folder: Contains development resources, such as images, data import maps, and other templates.

·         SampleCode Folder: Contains sample code to demonstrate how to develop by using Microsoft Dynamics 365.

·         Schemas Folder: Contains the schemas used for validation by using Microsoft Dynamics.

·         Tools Folder: Contains the tools needed to register plug-ins and workflows, and to create strongly typed classes.

·         LicenseTerms.docx: The Microsoft software license terms for this SDK.

·         Other Redistributable.txt: Lists the redistributable files for Microsoft Dynamics 365.

·         Third Party Notices for Dynamics 365 SDK.docx: Contains notices of third-party code included in this SDK.

·         Dynamics365SDK.chm: The help file that contains SDK documentation.

·         Microsoft_Dynamics_365_SDK_Readme.htm: This readme file.

·         EntityMetadata.xlsx: Contains all the metadata information for a default installation.

·         Message-entity support for plug-ins.xlsx: Contains the listing of plug-in support per entity and message.

Get Started

Before you use the Microsoft Dynamics 365 SDK, you should review the December 2016 update for Dynamics 365 (online and on-premises) Readme.
