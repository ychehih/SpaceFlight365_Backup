Microsoft Dynamics CRM 2015 SDK Readme

SDK Version 7.0.a

October 2014

This document provides important, late-breaking information about the Microsoft Dynamics CRM 2015 Software Development Kit (SDK) for:

·         Microsoft Dynamics CRM Online 2015 Update

·         Microsoft Dynamics CRM 2015 

 

Microsoft Dynamics CRM 2015 SDK Download Contents

The Microsoft Dynamics CRM 2015 SDK download package includes the following files and folders:

·         Bin Folder - Contains the assemblies needed for development by using Microsoft Dynamics CRM 2015 and Microsoft Dynamics CRM Online.

·         Resources Folder - Contains development resources, such as images, data import maps, and other templates.

·         SampleCode Folder - Contains sample code to demonstrate how to develop by using Microsoft Dynamics CRM 2015 and Microsoft Dynamics CRM Online.

·         Schemas Folder - Contains the schemas used for validation by using Microsoft Dynamics CRM 2015 and Microsoft Dynamics CRM Online.

·         Solutions Folder - Contains sample solutions for use with Microsoft Dynamics CRM 2015 and Microsoft Dynamics CRM Online.

·         Tools Folder - Contains the tools needed to register plug-ins and workflows, and to create strongly typed classes.

·         Walkthroughs Folder - Contains the completed solutions that correspond to walkthroughs presented in the SDK documentation.

·         LicenseTerms.docx - The Microsoft software license terms for this SDK.

·         Other Redistributable.txt  Lists the redistributable files for Microsoft Dynamics CRM 2015.

·         Third Party Notices for CRM 2015 SDK.docx  Contains notices of third party code included in this SDK.

·         CrmSdk2015.chm - The help file that contains SDK documentation.

·         Microsoft_Dynamics_CRM_2015_SDK_Readme.htm - This readme file.

·         EntityMetadata.xlsx  Contains all the metadata information for a default installation.

·         Message-entity support for plug-ins.xlsx  Contains the listing of plug-in support per entity and message.

Get Started

Before you use the Microsoft Dynamics CRM 2015 SDK, you should review the Microsoft Dynamics CRM Online 2015 Update and Microsoft Dynamics CRM 2015 Readme.

You should also read the Deploying and administering Microsoft Dynamics CRM, located on TechNet. Detailed instructions for installing Microsoft Dynamics CRM Server 2015 and Microsoft Dynamics CRM 2015 for Microsoft Office Outlook software are provided in that document.

Feature Area Known Issues

Developer toolkit not available

An updated Developer Toolkit for Microsoft Dynamics CRM is planned for this release but it is not ready at this time. Please refer to the CRM 2013 SDK for information about the Developer Toolkit for Microsoft Dynamics CRM 2013 and Microsoft Dynamics CRM Online. The CRM 2013 SDK is available for download: Microsoft Dynamics CRM Software Development Kit (SDK) for CRM Online and CRM 2013 (on-premises)

Developer extensions not available

Assemblies and utilities to support the SDK content under SDK extensions for Microsoft Dynamics CRM are not included in this release. The capabilities provided by this section are being reviewed, tested, and updated for Microsoft Dynamics CRM 2015 and Microsoft Dynamics CRM Online. When this work is complete, the supporting assemblies and utilities will be added back to the SDK.

CRM for Tablets

There are no issues at this time.
