This managed solution contains a JavaScript web resource: SDK.DuplicateDetection.js (sample_/duplicateDetection/scripts/SDK.DuplicateDetection.js) which includes a function that can be set in the OnSave event of main or quick create forms to implement duplicate detection checking when records on entities that are configured for duplicate detection are created.

SDK.DuplicateDetection.js depends other HTML web resource to provide the user interface to show any duplicate records that are discovered. This HTML web resource and a supporting JavaScript web resource must be created for each entity. This solution contains some examples that have been create for the Contact and Lead system entities as well as for a custom entity named Duplicate Detect Test.

For more information, see the ReadMe.docx included with the downloadable sample code in the SDK.
